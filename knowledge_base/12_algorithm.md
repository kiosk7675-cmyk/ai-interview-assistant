# 算法与数据结构面经

## 数据结构

### 数组与链表
数组：连续内存，O(1) 随机访问，O(n) 插入删除。链表：离散内存，O(n) 访问，O(1) 插入删除（已知位置）。双向链表可双向遍历。循环链表尾节点指向头节点。常见题：反转链表、检测环（快慢指针）、合并有序链表、找中间节点。

### 栈与队列
栈：后进先出（LIFO），函数调用栈、括号匹配、表达式求值。队列：先进先出（FIFO），BFS、任务调度。双端队列（Deque）：两端均可入出，滑动窗口最大值。优先队列/堆：O(log n) 插入和取极值，Top-K 问题。

### 哈希表
基于哈希函数将 key 映射到桶位。冲突解决：链地址法（开链）、开放寻址法（线性探测/二次探测）。负载因子 = 元素数/桶数，超过阈值扩容 rehash。平均 O(1) 查找，最坏 O(n)。LRU Cache：哈希表 + 双向链表，O(1) get/put。

### 树
二叉树遍历：前序（根左右）、中序（左根右）、后序（左右根）、层序（BFS）。二叉搜索树（BST）：中序遍历有序，平均 O(log n) 查找，最坏 O(n) 退化为链表。平衡二叉树（AVL）：左右子树高度差 ≤1，旋转调整。红黑树：近似平衡，插入删除最多 3 次旋转，Java TreeMap/HashMap 用。

### 图
表示：邻接矩阵（空间 O(V²)）、邻接表（空间 O(V+E)）。遍历：DFS（递归/栈，检测环、拓扑排序）、BFS（队列，最短路径无权图）。最短路径：Dijkstra（正权，O(E log V)）、Bellman-Ford（可有负权）、Floyd（多源）。最小生成树：Prim、Kruskal（并查集）。

## 排序算法

| 算法 | 平均 | 最坏 | 空间 | 稳定 |
|------|------|------|------|------|
| 冒泡排序 | O(n²) | O(n²) | O(1) | 稳定 |
| 选择排序 | O(n²) | O(n²) | O(1) | 不稳定 |
| 插入排序 | O(n²) | O(n²) | O(1) | 稳定 |
| 归并排序 | O(n log n) | O(n log n) | O(n) | 稳定 |
| 快速排序 | O(n log n) | O(n²) | O(log n) | 不稳定 |
| 堆排序 | O(n log n) | O(n log n) | O(1) | 不稳定 |

快排：选基准（pivot），小于放左大于放右，递归。优化：三数取中、小数组切插入排序。归并：分治+合并，稳定排序首选。堆排序：建大根堆 O(n)，逐个弹出 O(n log n)。

## 高频算法题型

### 双指针
对撞指针：有序数组两数之和、回文判断。快慢指针：链表找环、找中点。滑动窗口：最长/最短子串，维护窗口内状态。

### 二分查找
有序数组中查找，O(log n)。变形：查找第一个等于/最后一个等于/第一个大于等于。旋转有序数组查找。注意：`mid = left + (right - left) / 2` 防溢出。

### 动态规划
核心：最优子结构 + 重叠子问题 + 状态转移方程。步骤：定义状态 → 推导转移方程 → 确定初始值 → 确定遍历顺序。经典题：爬楼梯、背包问题（01/完全）、最长递增子序列、编辑距离、零钱兑换、股票买卖。

### 回溯
本质是 DFS + 剪枝。排列（顺序相关）、组合（顺序无关）、子集。模板：选择 → 递归 → 撤销选择。剪枝：排序后跳过重复、提前终止。

### 贪心
局部最优 → 全局最优。适用条件：贪心选择性质 + 最优子结构。经典：活动选择、跳跃游戏、区间调度、Huffman 编码。

### 字符串算法
KMP：模式串匹配 O(n+m)，next 数组记录最长相同前后缀。Rabin-Karp：哈希滚动匹配。Manacher：最长回文子串 O(n)。Trie 树：前缀匹配，O(L) 查找。

### 位运算技巧
`n & (n-1)` 去掉最低位 1（计算 1 的个数）。`n & (-n)` 取最低位 1（树状数组）。异或：`a ^ a = 0`，找唯一出现一次的数。两数交换：`a ^= b; b ^= a; a ^= b`。

## 复杂度分析

时间复杂度排序：O(1) < O(log n) < O(n) < O(n log n) < O(n²) < O(2ⁿ) < O(n!)。空间复杂度：额外使用的空间，递归深度计入栈空间。主定理分析分治递推式：T(n) = aT(n/b) + f(n)。
